O back-end é a parte do sistema de software que nunca é visualizada de fato pelo usuário final, mas que é responsável por tarefas essenciais para que o sistema funcione, tais como processamento de dados, gerenciamento de banco de dados, comunicação com outros sistemas e execução de regras de negócio.
